# Hyperlocal Emergency and Disaster Alert Network

A citizen-powered platform for real-time hyperlocal emergency alerts with AI-based fake alert detection.

## Project Structure
- frontend/ — React PWA (Vite)
- backend/ — Node.js + Express + MongoDB
- ai-service/ — Python Flask (Custom AI Algorithm)

## Features
- Real-time emergency alert reporting
- GPS-based hyperlocal alert propagation
- Custom Weighted Adaptive Scoring Algorithm
- Authority dashboard with verify/dismiss/resolve
- Progressive Web App — installable on any phone
- Grievance system for false alert reporting

## Tech Stack
Frontend: React, Vite, Leaflet.js, Socket.io-client
Backend: Node.js, Express, MongoDB, Socket.io
AI Service: Python, Flask (zero ML libraries)
Database: MongoDB Atlas
